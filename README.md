# React Testing

Testing React/Redux application with Mocha, Chai, and Chai-Jquery

## Development Setup 

```
> git clone https://github.com/pporche87/react-testing.git
> cd react-testing
> npm install
> npm start
```
## Technical Stack 
- React/Redux
- Webpack, Webpack Dev Server
- Babel
- JavaScript, jQuery, Lodash
- Mocha, Chai
- jsdom 
- Node
